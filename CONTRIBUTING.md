# Contributing to Invizible Tracker Control Pro

Thanks for your interest!

## Current status
This project is an early-stage architecture and skeleton.
Major components are not yet production-ready.

## How you can help
- VPNService packet parsing
- Tracker detection & filtering
- DNSCrypt integration
- Tor / I2P backend integration
- UI improvements (Jetpack Compose)
- Security reviews

## Rules
- No telemetry
- No logging of user traffic
- Respect privacy-by-design principles
- One VPNService only

## Submitting changes
- Fork the repo
- Create a feature branch
- Submit a Merge Request / Pull Request
