# Code of Conduct

## Our Pledge
We aim to foster an open, respectful, and harassment-free environment for
everyone participating in the Invizible Tracker Control Pro project.

## Expected Behavior
- Be respectful and constructive
- Assume good faith
- Accept constructive criticism
- Focus on technical merit and privacy principles
- Respect different skill levels and backgrounds

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks
- Trolling or intentionally disruptive behavior
- Publishing private information without consent
- Pressuring maintainers or contributors

## Project Values
- Privacy by design
- Transparency
- Security over convenience
- No telemetry, no tracking, no hidden data collection

## Enforcement
Project maintainers have the right to remove, edit, or reject contributions
that violate this code of conduct.

## Scope
This code applies to all project spaces, including issues, merge requests,
pull requests, discussions, and community channels.

## Reporting
If you experience or witness unacceptable behavior, please open a private
issue or contact the maintainers.

---
This Code of Conduct is intentionally minimal and focused on mutual respect.
